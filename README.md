# 🛡️ Crisis Navigation System (CrisisNav) - Re-envisioned

**CrisisNav** is a premium, industry-grade emergency guidance platform. This version features a complete mobile-first redesign based on high-fidelity designs and a real Node.js/Express backend integration.

---

## ✨ Features
- **💎 Premium Mobile UI:** Fully responsive, glassmorphism-based design with vibrant category-specific themes.
- **🧭 Dynamic Step Navigation:** Real-time tracking of emergency SOPs with clear visual progress.
- **🚀 Node.js Backend:** Centralized data management for emergency protocols and action logging.
- **📱 Touch-Optimized:** Designed for rapid action on mobile devices during critical events.

---

## 🛠️ Technology Stack
- **Frontend:** Vanilla HTML5, CSS3 (Custom Properties, Grid/Flex), JavaScript (Fetch API).
- **Backend:** Node.js, Express, CORS.
- **Icons:** Phosphor Icons.

---

## 🏃 How to Run Locally

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Start the Server:**
   ```bash
   npm start
   ```

3. **Open in Browser:**
   Navigate to `http://localhost:3000`
